# spring Annotation
### 의존 주입
객체의 의존성을 가지는 부분에 사용한다.<br/>
- `@Autowired`
    - 타입으로 의존성을 주입한다.
    - 변수, 생성자, setter, 일반 메서드 등에 적용 가능
- `@Resource`
    - 이름으로 의존성을 주입한다.
    - 변수, setter 등에 적용 가능
- `@Qualifier`
    - 연결할 빈의 값 타입을 지정한다.
